# 🏥 Hệ thống Quản lý và Truy xuất Hồ sơ Bệnh án

## 📌 Giới thiệu
Hệ thống quản lý và truy xuất hồ sơ bệnh án ứng dụng công nghệ Blockchain nhằm đảm bảo tính minh bạch, bảo mật và toàn vẹn dữ liệu y tế.
Hệ thống cho phép lưu trữ, truy xuất và xác thực thông tin bệnh án giữa các bên liên quan một cách an toàn và nhanh chóng.

---

## ⚙️ Công nghệ sử dụng

- **Frontend**: ReactJS
- **Backend**: NodeJS + ExpressJS
- **Database**: MySQL/Mongodb
- **Blockchain**: Ethereum Sepolia Testnet
- **Smart Contract**: Solidity
- **Blockchain Framework**: Hardhat
- **Blockchain Library**: ethers.js

---
## ⚙️ Cài đặt dự án

📌 Frontend
```bash
cd frontend
npm install
npm start
📌 Backend
cd backend
npm install
npm run dev
📌 Blockchain
cd blockchain
npm install
npx hardhat compile
npx hardhat run scripts/deploy.js --network sepolia
